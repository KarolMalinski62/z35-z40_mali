1. kiedy powstał JavaScript (podaj datę).

    w 1995 roku

      2. dlaczego jest to język skryptowy

    JavaScript jest przede wszystkim językiem skryptowym - to znaczy interpretowanym. Nie musi
    zostać skompilowany do kodu maszynowego, aby można było zobaczyć efekty jego działania.
    Wystarczy nam do tego przeglądarka internetowa, która ten język obsługuje.

      3. gdzie jest wykonywany JS (klient lub serwer)

    Jest wykonywany na komputerze klienta

      4. dlaczego jest JS to bezpieczne narzędzie.

    JavaScript jest stosunkowo bezpiecznym narzędziem, ponieważ działa w izolowanym
    środowisku przeglądarki, ma ograniczone uprawnienia oraz działa zgodnie z zasadą Same
    Origin Policy. Jednakże, nadal może być używany w sposób niebezpieczny, dlatego ważne
    jest, aby programiści przestrzegali najlepszych praktyk programowania.

      5. czy jest to język obiektowy , jeśli tak to dlaczego.

    Tak, JavaScript jest językiem programowania zorientowanym obiektowo. Pozwala on na
    tworzenie obiektów, które zawierają dane i funkcje, co umożliwia programistom organizację i
    zarządzanie kodem w sposób modularny oraz reużywalny.

      6. Wymień zalety JS:

    Zaleta 1: Wysoka interaktywność - JavaScript umożliwia dynamiczną manipulację zawartością strony, co pozwala na interaktywne i atrakcyjne interfejsy użytkownika.

    Zaleta 2: Uniwersalność - Jest wspierany przez większość przeglądarek internetowych, co oznacza, że można go stosować na różnych platformach.

    Zaleta 3: Szeroka społeczność i bogata biblioteka - Istnieje duże wsparcie społeczności programistycznej oraz liczne biblioteki i frameworki ułatwiające pracę z JavaScript.

      7. Framework to:

    Framework to zestaw narzędzi, bibliotek i szablonów, które ułatwiają i przyspieszają proces tworzenia aplikacji lub projektów informatycznych, zapewniając gotowe rozwiązania i struktury.

      8. Do aplikacji mobilnych można użyć Framework:

    Jednym z popularnych frameworków do tworzenia aplikacji mobilnych jest React Native, który pozwala na pisanie aplikacji mobilnych używając JavaScriptu i Reacta.

      9. Aplikacje responsywne to:

    Aplikacje responsywne to strony lub aplikacje, które automatycznie dostosowują się do różnych rozmiarów ekranów i urządzeń, zapewniając optymalne doświadczenie użytkownika zarówno na komputerach, tabletach, jak i smartfonach.

      10. Przepisz linie kodu pod nimi wytłumaczenie jak działa ten fragment kodu:

    <input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')">

    Ta linia kodu tworzy przycisk na stronie internetowej. Po kliknięciu przycisku, zostanie wywołana funkcja `WinOpen(' ')`. Jednakże, funkcja `WinOpen` nie jest dostarczona w tym fragmencie kodu, więc jej działanie nie zostało zdefiniowane.

      11. Przepisz linie kodu pod nimi wytłumaczenie jak działa ten fragment kodu:

    <script>
    function openWindow() {
    window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
    }
    </script>
    <button onclick="openWindow()">Otwórz nowe okno</button>

    Ta linia kodu otwiera nowe okno przeglądarki, które wyświetli zawartość strony "obraz.html". Dodatkowe parametry (np. "toolbar=no") ustawiają różne opcje wyglądu i zachowania nowego okna.

    toolbar=no

    Parametr "toolbar=no" oznacza, że pasek narzędziowy przeglądarki (który zawiera przyciski takie jak "Wstecz", "Dalej", "Odśwież", itp.) nie będzie widoczny w nowym oknie.

    directories=no

    Parametr "directories=no" oznacza, że przyciski nawigacyjne do katalogów nie będą widoczne w nowym oknie.

    menubar=no

    Parametr "menubar=no" oznacza, że pasek menu (zwykle zawierający opcje takie jak "Plik", "Edycja", itp.) nie będzie widoczny w nowym oknie.

    height=280

    Parametr "height=280" ustawia wysokość nowego okna na 280 pikseli.

    width=160

    Parametr "width=160" ustawia szerokość nowego okna na 160 pikseli.

    top=200

    Parametr "top=200" ustawia pozycję górnego brzegu nowego okna na 200 pikseli od góry ekranu.

    left=200

    Parametr "left=200" ustawia pozycję lewego brzegu nowa okna na 200 pikseli od lewej krawędzi ekranu.

      12. Opisz następującą instrukcję window.close()

    Instrukcja `window.close()` w JavaScript służy do zamknięcia aktywnego okna
    przeglądarki. Jednakże, ze względów bezpieczeństwa, większość przeglądarek może
    zablokować automatyczne zamknięcie okna bez wyraźnej interakcji użytkownika.
