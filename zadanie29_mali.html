Odpowiedzi na pytania teoretyczne

1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

Sposób 1: Używając zapisu postaci: nazwa_obiektu.nazwa_właściwości = wartość;
Sposób 2: Używając zapisu postaci: nazwa_obiektu['nazwa_właściwości'] = wartość;

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Do właściwości i metod można się odwołać poprzez nazwę obiektu, kropkę oraz nazwę pola lub metody. Na przykład: nazwa_obiektu.nazwa_właściwości lub nazwa_obiektu.nazwa_metody().

3. Wymień cechy charakterystyczne konstruktora.

- Jest wywoływany automatycznie w chwili tworzenia obiektu danej klasy.
- Nazywa się tak samo jak klasa.
- Nie zwraca żadnej wartości.
- Może przyjmować argumenty.
- Służy do inicjalizacji obiektu.

4. Konstruowanie obiektu na podstawie przykładu - obiekt ma trzy pola oraz jedną metodę.

Poniżej znajduje się przykład konstruktora obiektu z trzema polami (nazwisko, imie, zawod) oraz jedną metodą (wyświetl):

var osoba_1 = {
 nazwisko: 'Kowalski',
 imie: 'Jan',
 zawod: 'piekarz',
 wyświetl: function() {
  Document.write(this.nazwisko + ' ' + this.imie)
 }
}

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

Słowo kluczowe this wskazuje na aktualny obiekt. Pola i metody oddzielamy za pomocą kropki. Na przykład: nazwa_obiektu.nazwa_pola lub nazwa_obiektu.nazwa_metody().

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

Poniżej znajduje się przykład konstruktora w języku JavaScript:

function klient(nazwisko_k, imie_k, zawod_k) {
 this.nazwisko = nazwisko_k;
 this.imie = imie_k;
 this.zawod = zawod_k;
 this.wypisz = function() {
  alert(this.nazwisko + ' ' + this.imie);
 }
}

7. Znaczenie słowa kluczowego New wraz z przykładem.

Słowo kluczowe new służy do tworzenia nowych obiektów na podstawie konstruktora. Poniżej znajduje się przykład jego użycia:

var osoba1 = new klient('Kowalski', 'Jan', 'kierowca');
var osoba2 = new klient('Nowak', 'Anna', 'sekretarka');

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

Słowo kluczowe prototype jest używane w JavaScript do definowania metod i właściwości dla obiektu. Poniżej znajduje się przykład jego użycia:

function klient() {
 this.nazwisko = 'Bielski';
 this.imie = 'Paweł';
}
klient.prototype.pisz_dane = function () {
 document.write(this.nazwisko + ' ' + this.imie);
}
klient.prototype.zawod = 'kierowca';
var osoba1 = new klient();
osoba1.pisz_dane();

Właściwość prototype pozwala na dodanie metod i właściwości do istniejącego już konstruktora. Metody i właściwości dodane za pomocą prototype są współdzielone przez wszystkie instancje danego konstruktora.
